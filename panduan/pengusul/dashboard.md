# Dashboard Pengusul

## Posisi Anda dalam perjalanan program

Sebagai Pengusul, Anda membawa kegiatan dari **ide** menjadi **luaran**. Urutannya: siapkan proposal → ajukan → tanggapi revisi bila ada → jalankan kegiatan yang diterima → kirim laporan dan luaran sesuai jadwal.

Untuk melihat gambaran besar sebelum memakai menu, baca [Alur End-to-End & 5W1H IRIS](/alur-proposal). Halaman ini selanjutnya menjelaskan tempat Anda memantau perjalanan tersebut.

Setelah login, Pengusul akan diarahkan ke halaman **Dashboard Pengusul**.

## Apa yang Ada di Dashboard
![Dashboard Pengusul](/pengusul/dashboard-pengusul.png)
Dashboard Pengusul menampilkan:

- **Ringkasan proposal Anda** — jumlah proposal aktif, yang sedang diproses, diterima, dan ditolak
- **Status Kelengkapan Usulan** - status apakah usulan penelitian dan usulan PKM sudah diunggah 
- **Kalender periode aktif** — jadwal periode pengajuan yang sedang berjalan, lengkap dengan fase pengajuan, review, Monitoring & Evaluasi, dan diseminasi

| Dashboard | Halaman utama |

## Cara Login

IRIS menggunakan **SSO (Single Sign-On) universitas** sebagai metode login utama. Gunakan akun email institusi Anda.

Jika SSO tidak tersedia, Anda bisa login menggunakan email dan password langsung di halaman login.

::: tip Punya lebih dari satu peran?
Jika akun Anda memiliki lebih dari satu peran, sistem akan menampilkan halaman **Pilih Peran** setelah login. Pilih **Pengusul** untuk masuk ke dashboard ini. Anda bisa berpindah peran kapan saja melalui menu di sudut kanan atas.
:::
