# Penelitian & PKM Saya

Setelah proposal diterima, penelitian Anda masuk ke fase pelaksanaan. Di sini Anda wajib mengunggah dokumen untuk setiap tahapan dan mengunci pengiriman agar reviewer bisa menilai.

## Mengakses Halaman Penelitian
Dari sidebar, pilih **Penelitian Saya** atau **PKM Saya** sesuai jenis proposal Anda. Daftar menampilkan semua penelitian aktif beserta statusnya.
![](/pengusul/kegiatanBerlangsung-pengusul.png)
Klik detail penelitian dengan ikon (👁️) untuk membuka halaman detail yang menampilkan semua tahapan dalam bentuk tab.
![](/pengusul/detailKegiatan-pengusul.png)


## Tahapan Penelitian

Setiap penelitian melewati tahapan berikut secara berurutan:

| Tahapan | Keterangan |
|---------|-----------|
| **Proposal** | Tahap awal — dokumen & penilaian proposal (hanya bisa dilihat, tidak bisa upload ulang) |
| **Laporan Kemajuan** | Laporan perkembangan penelitian di tengah periode |
| **Monev** | Monitoring & Evaluasi — dinilai oleh Reviewer 1 |
| **Laporan Akhir** | Laporan hasil penelitian secara lengkap |
| **Cek Plagiarisme** | Upload hasil cek plagiarisme beserta persentasenya |
| **Diseminasi** | Upload materi presentasi atau tautan video diseminasi |
| **Publikasi** | Upload bukti publikasi jurnal/artikel |

---

## Mengunggah Dokumen per Tahapan

Untuk setiap tahapan (kecuali Proposal):

1. Buka tab tahapan yang sesuai
2. Klik **Upload File**
![](/pengusul/uploadTahapan-pengusul.png)
3. Pilih file dari perangkat Anda

### Ketentuan File

| Tahapan | Format yang Diterima | Ukuran Maks |
|---------|---------------------|------------|
| Laporan Kemajuan | PDF | 50 MB |
| Monev | PDF | 50 MB |
| Laporan Akhir | PDF | 50 MB |
| Cek Plagiarisme | PDF | 50 MB |
| Diseminasi | PDF, PPT, PPTX | 50 MB |
| Publikasi | PDF | 50 MB |

### Khusus Cek Plagiarisme

Saat upload, Anda wajib mengisi **persentase plagiarisme** (angka 0–100) yang tertera di laporan hasil cek. Kolom ini wajib diisi.

### Khusus Publikasi

Saat upload, Anda wajib mengisi:
- **Judul Artikel**
- **Nama Jurnal**
- **Indeks Jurnal** (contoh: Sinta 2, Scopus, dll.)
- **File Bukti Bayar**
![](/pengusul/publikasi-pengusul.png)

## Status File: Draft vs Submitted


Setiap file yang diunggah awalnya berstatus **Draft** — artinya hanya Anda yang bisa melihat dan masih bisa dihapus.
![](/pengusul/statusUpload-pengusul.png)

Setelah yakin semua file sudah benar, klik **Submit** untuk mengunci pengiriman:
![](/pengusul/submit-pengusul.png)

::: warning Perhatian sebelum Submit
- File yang sudah di-submit **tidak bisa dihapus atau diganti** (kecuali Diseminasi)
- Reviewer baru bisa mengakses dan menilai file setelah Anda submit
- Pastikan semua file sudah lengkap sebelum klik Submit
:::

### Menghapus File Draft

Selama masih berstatus **Draft**, file bisa dihapus dengan klik ikon 🗑️ di baris file yang bersangkutan.
![](/pengusul/hapusDraf-pengusul.png)

## Melihat Hasil Penilaian

Setelah reviewer menilai suatu tahapan, skor akan muncul di tab tahapan tersebut:
![](/pengusul/hasilPenilaian-pengusul.png)
- **Skor per kriteria** — nilai yang diberikan reviewer untuk setiap kriteria
- **Skor total** — perhitungan tertimbang dari semua kriteria
- **Catatan reviewer** — komentar tambahan dari reviewer (jika ada)

::: info Penilaian dikunci
Setelah reviewer menyimpan penilaian, Anda tidak bisa mengupload file baru ke tahapan yang sudah dinilai. Tahapan tersebut dianggap selesai.
:::
