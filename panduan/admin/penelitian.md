# Memantau Penelitian & PKM

Setelah proposal diterima, Admin dapat memantau seluruh progres penelitian yang sedang berjalan — mulai dari tahapan Laporan Kemajuan hingga Publikasi.

## Mengakses Daftar Penelitian

Dari sidebar klik **Pelaksanaan**, lalu klik **Penelitian** atau **Pengabdian (PKM)** untuk melihat semua kegiatan aktif pada periode yang sedang dibuka.

Daftar menampilkan:
- Judul proposal
- Nama pengusul
- Kategori (Wajib / Pilihan)
- Periode aktif
- Status penelitian (Aktif / Selesai / Dibatalkan)
![](/admin/daftarPenelitian-admin.png)
---

## Membuka Detail Penelitian
![](/admin/detailPenelitian-admin.png)
Klik ikon 👁️ di baris penelitian/pkm untuk membuka halaman detail. Halaman ini menampilkan semua tahapan dalam bentuk tab:
![](/admin/tahapan-admin.png)
| Tab | Isi |
|-----|-----|
| **Proposal** | Ringkasan penilaian proposal awal beserta skor per reviewer |
| **Laporan Kemajuan** | File yang diupload pengusul + skor reviewer |
| **Monev** | File Monev + skor reviewer |
| **Laporan Akhir** | File laporan akhir + skor reviewer |
| **Cek Plagiarisme** | Dokumen & persentase plagiarisme |
| **Diseminasi** | Materi atau tautan diseminasi + skor reviewer |
| **Publikasi** | Bukti publikasi jurnal |

---

## Memantau Skor per Tahapan

Di setiap tab, Admin bisa melihat:

- **Skor per reviewer** — skor masing-masing reviewer yang sudah menilai (ternormalisasi ke skala 0–100)
- **Rata-rata skor** — rata-rata dari semua reviewer yang sudah menilai
- **Catatan reviewer** — komentar yang ditulis reviewer saat penilaian
![](/admin/hasilPenilaian-admin.png)
::: info Skor ternormalisasi
Skor yang ditampilkan sudah dikonversi ke skala 0–100 berdasarkan bobot kriteria masing-masing tahapan, sehingga mudah dibandingkan antar tahapan dan antar penelitian.
:::

---

## Mengelola Data Publikasi

Khusus tab **Publikasi**, Admin dapat memperbarui informasi artikel jurnal yang diupload Pengusul:

1. Klik **Edit Informasi Publikasi** di baris file publikasi
2. Perbarui kolom:
   - **Judul Artikel**
   - **Nama Jurnal**
   - **Indeks Jurnal** (Sinta 1, Sinta 2, Scopus, dll.)
3. Klik **Simpan**

::: tip Kapan fitur ini digunakan?
Gunakan fitur ini jika Pengusul mengisi informasi publikasi yang salah atau perlu dikoreksi setelah proses verifikasi oleh Admin.
:::

---

## Mengunduh/View Dokumen

Di setiap tab tahapan, Admin bisa melihat file yang disubmit Pengusul dengan klik tombol **view/unduh** di samping nama file.

## Status Penelitian
![](/admin/status-admin.png)
| Status | Artinya |
|--------|---------|
| `aktif` | Penelitian sedang berjalan |
| `selesai` | Semua tahapan sudah diselesaikan |
| `dibatalkan` | Penelitian dibatalkan |
