# Kriteria & Rubrik Penilaian

Setiap periode memiliki kriteria penilaian sendiri yang bisa dikonfigurasi secara bebas. Kriteria ini yang digunakan Reviewer saat menilai proposal.
![Akses Kriteria](/admin/lihatKriteria-admin.png)
Akses melalui **Sidebar → Periode → [pilih periode] → Detail**.

## Menambah Rubrik Baru

1. Pada **Detail** Klik **Tambah**
   ![Tambah Kriteria](/admin/tambahKriteria-admin.png)
2. Isi nama kriteria dan nilai bobot (dalam persen)
   ![Form Kriteria](/admin/formKriteria-admin.png)
3. Atur urutan tampilan jika diperlukan
4. Klik **Simpan**

::: tip Rubrik wajib diisi di setiap kategori
Pastikan seluruh kategori sudah memiliki rubrik agar proposal dapat dinilai.
:::

## Mengubah Kriteria

![Edit Kriteria](/admin/editKriteria-admin.png)

1. Klik ikon edit (✏️) pada baris kriteria
2. Ubah nama atau bobot
3. Klik **Simpan Perubahan**

## Menghapus Kriteria

![Hapus Kriteria](/admin/hapusKriteria-admin.png)
Klik ikon hapus (🗑️) lalu konfirmasi. Kriteria yang sudah digunakan dalam penilaian tidak dapat dihapus.

---

## Rubrik Penilaian

Setiap kriteria memiliki **rubrik** — yaitu panduan deskripsi untuk setiap nilai skor dari 1 sampai 5.

### Cara Mengatur Rubrik

1. Klik nama kriteria untuk membuka detailnya
   ![lihat Rubrik](/admin/lihatRubrik-admin.png)
2. Pada bagian **Rubrik**, isi deskripsi untuk masing-masing skor, **contoh pengisian**:
   ![isi rubrik](/admin/formRubrik-admin.png)

::: tip Klik salah satu rubrik untuk melihat panduan pengisian.
:::

## Menyalin Kriteria dari Periode Lain

Jika kriteria periode baru sama dengan periode sebelumnya:

1. Di halaman Kriteria, klik **Copy**
![Copy Kriteria](/admin/copyKriteria-admin.png)
2. Pilih periode sumber dari daftar
![Copy Kriteria](/admin/formCopyKriteria-admin.png)
3. Klik **Copy Kriteria**

Semua kriteria dan rubrik dari periode sumber akan disalin. Anda tetap bisa mengubahnya setelah penyalinan.

## Import Data

Jika ingin memasukkan data (Proposal, Anggota, Review, dan Penilaian) secara massal khusus untuk periode yang dipilih:

1. Di halaman Detail Periode, **klik Import Data**
![Import Data](/admin/importData-admin.png)
2. Download template jika belum punya dan isi dengan data yang diperlukan sistem
![Form Import Data](/admin/formImportData-admin.png)
3. Upload file dengan format .xls atau .xlsx
4. Klik **Mulai proses import**

Proses ini akan melakukan import kepada seluruh data yang berada di spreadsheet.
