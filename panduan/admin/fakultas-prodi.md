# Kelola Fakultas & Program Studi

Data Fakultas dan Program Studi digunakan saat mendaftarkan pengguna dan pengajuan proposal. Pastikan data ini sudah lengkap sebelum membuka periode pengajuan.

## Kelola Fakultas

![Fakultas](/admin/fakultas-admin.png)
Akses melalui **Sidebar → Akademik → Fakultas**.

### Menambah Fakultas

![Tambah Fakultas](/admin/tambahFakultas-admin.png)

1. Klik **Tambah Fakultas**
2. Isi nama fakultas
3. Klik **Simpan**

### Mengubah Nama Fakultas

![Edit fakultas](/admin/editFakultas-admin.png)

1. Klik ikon edit (✏️) pada baris fakultas
2. Ubah nama
3. Klik **Simpan**

### Menghapus Fakultas

![Hapus Fakultas](/admin/hapusFakultas-admin.png)
Klik ikon hapus (🗑️) lalu konfirmasi. Fakultas yang masih memiliki Program Studi atau pengguna aktif tidak dapat dihapus.

---

## Kelola Program Studi

![Prodi Admin](/admin/prodi-admin.png)
Akses melalui **Sidebar → Akademik → Program Studi**.

### Menambah Program Studi

![Menambah Prodi](/admin/tambahProdi-admin.png)

1. Klik **Tambah Program Studi**
2. Isi nama program studi
3. Pilih **Fakultas** induk dari daftar
4. Klik **Simpan**

### Mengubah Program Studi

![Edit Prodi](/admin/editProdi-admin.png)

1. Klik ikon edit (✏️)
2. Ubah nama atau fakultas induk
3. Klik **Simpan**

### Menghapus Program Studi

![Hapus Prodi](/admin/hapusProdi-admin.png)
Klik ikon hapus (🗑️) lalu konfirmasi. Program studi yang masih memiliki pengguna aktif tidak dapat dihapus.

::: info Urutan pengaturan yang disarankan
Selalu buat **Fakultas** terlebih dahulu sebelum membuat Program Studi, karena Program Studi membutuhkan Fakultas sebagai induk.
:::
