# Dashboard Admin

Setelah login, Admin akan langsung diarahkan ke halaman **Dashboard Admin**.

## Dashboard Admin
![Gambar Dashboard Admin](/admin/dashboard-admin.png)

Dashboard Admin menampilkan ringkasan kondisi sistem secara keseluruhan:

- **Statistik** — jumlah proposal masuk, sedang diproses, diterima, dan ditolak
- **Kalender periode aktif** — menampilkan jadwal fase pengajuan, review, monitoring & evaluasi, serta diseminasi menggunakan FullCalendar

## Navigasi Sidebar
![Navigasi Admin](/admin/sidebar-admin.png)
Dari sidebar, Admin dapat mengakses:

| Menu | Fungsi |
|------|--------|
| Dashboard | Kembali ke halaman utama |
| Pengguna | Kelola akun dan peran pengguna |
| Periode | Buat dan atur periode pengajuan |
| Proposal | Daftar semua proposal & verifikasi |
| Fakultas | Kelola data fakultas |
| Program Studi | Kelola data program studi |

## Langkah Berikutnya

Setelah login pertama kali, langkah yang dianjurkan:

1. Pastikan data **Fakultas** dan **Program Studi** sudah lengkap → [Kelola Fakultas & Prodi](/panduan/admin/fakultas-prodi)
2. Buat **Periode** pengajuan → [Kelola Periode](/panduan/admin/periode)
3. Atur **Kriteria & Rubrik** penilaian → [Kriteria & Rubrik](/panduan/admin/kriteria)
4. Pastikan akun **Pengguna** sudah terdaftar dan memiliki peran yang benar → [Kelola Pengguna](/panduan/admin/pengguna)

Setelah itu, sistem siap menerima pengajuan proposal dari Pengusul.
